Introducción a la web

¿Qué es la World Wide Web?

La World Wide Web (WWW) es un sistema que permite publicar, acceder y relacionar información a través de Internet mediante páginas y aplicaciones web.

Es importante no confundir ambos conceptos:

  • Internet es la red que conecta computadoras y otros dispositivos entre sí
  • La Web es uno de los servicios que funciona sobre Internet
  • Para usar la Web normalmente se emplea un navegador
  • Los recursos web se identifican mediante una URL

En términos simples, la Web permite que una persona, desde un celular, una notebook o una PC, solicite información a un servidor y la visualice en forma de texto, imágenes, audio, video o interfaces interactivas.

¿Cómo funciona la web?

Cuando una persona abre un sitio web, ocurre una secuencia de pasos entre el cliente, la red y el servidor.

  1. Se escribe una dirección web o se hace clic en un enlace
  2. El navegador interpreta esa dirección como una URL
    • La URL indica a qué recurso se quiere acceder
    • Por ejemplo: https://www.ejemplo.com/productos
  3. La computadora necesita averiguar a qué máquina conectarse
    • Cada equipo conectado a Internet tiene una dirección IP
    • Como las personas recuerdan mejor nombres que números, se usan nombres de dominio como www.ejemplo.com
    • Los DNS (Domain Name System) traducen ese nombre a una IP
  4. Con la IP obtenida, el navegador se comunica con el servidor y envía una solicitud HTTP
  5. El servidor recibe la solicitud, la procesa y responde
    • Puede indicar que el recurso no existe
    • Puede redirigir al cliente a otra dirección
    • Puede devolver archivos ya existentes, como HTML, CSS, imágenes o videos
    • Puede ejecutar un programa para generar una respuesta dinámica
  6. El navegador interpreta la respuesta y construye la página que ve la persona usuaria
    • Normalmente combina HTML, CSS, JavaScript y otros recursos
    • Distintos navegadores pueden mostrar pequeñas diferencias

En resumen, la Web funciona mediante un modelo de pedido y respuesta: el cliente solicita un recurso y el servidor responde.

Lado del cliente

El lado del cliente reúne todo lo que ocurre en el dispositivo de la persona usuaria una vez que recibe la respuesta del servidor.

  • Se ejecuta principalmente en el navegador
  • Incluye la estructura, la presentación y parte del comportamiento de la página
  • Intervienen tecnologías como HTML, CSS y JavaScript
  • También se lo conoce como front-end

El objetivo del lado del cliente es mostrar información e interactuar con la persona usuaria de forma clara, rápida y cómoda.

Lado del servidor

El lado del servidor agrupa los procesos que ocurren en una computadora remota encargada de recibir solicitudes y devolver respuestas.

  • Atiende solicitudes enviadas por clientes
  • Puede validar datos, consultar bases de datos o ejecutar lógica de negocio
  • Puede exponer servicios como APIs, autenticación o almacenamiento
  • También se lo conoce como back-end

En muchos sistemas, el cliente y el servidor trabajan en conjunto: el cliente presenta la interfaz y el servidor administra la información y las reglas del sistema.

Tecnologías del lado del cliente

Navegadores

El navegador es el programa que solicita recursos web, interpreta su contenido y lo muestra en pantalla. Además, administra pestañas, historial, favoritos, seguridad, descargas y herramientas de desarrollo.

Algunos navegadores populares son:

  • Chrome
  • Edge
  • Firefox
  • Brave
  • Safari
  • Opera

Aunque todos cumplen una función similar, no siempre interpretan exactamente igual todos los estándares. Por eso, al desarrollar sitios web, es importante probar en más de un navegador.

HTML

HTML significa HyperText Markup Language. Es el lenguaje de marcado que se utiliza para definir la estructura de una página web.

Con HTML se organizan los distintos elementos del contenido, por ejemplo:

  • títulos y subtítulos
  • párrafos y enlaces
  • imágenes, listas y tablas
  • formularios, botones y secciones

HTML no se encarga del diseño visual detallado ni de la lógica compleja. Su rol principal es describir qué es cada parte del contenido y cómo se relaciona con las demás.

La versión actual del estándar es conocida como HTML5, que incorporó etiquetas semánticas y soporte mejorado para multimedia y formularios.

Ejemplo de HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    El contenido de la página va aquí
  </body>
</html>

CSS

CSS significa Cascading Style Sheets. Es el lenguaje que se utiliza para definir la presentación visual de un documento HTML.

Con CSS se controlan aspectos como:

  • colores
  • tipografías
  • tamaños y espaciados
  • bordes, fondos y posiciones
  • distribución de elementos en la pantalla

Gracias a CSS es posible separar el contenido de su presentación. Esto facilita el mantenimiento, mejora la reutilización y permite adaptar un mismo contenido a diferentes dispositivos.

La denominación "en cascada" hace referencia a que varias reglas pueden afectar a un mismo elemento, y el navegador debe determinar cuáles tienen prioridad.

Ejemplo de CSS

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}
 
h1 {
  color: #0066cc;
  text-align: center;
}
 
p {
  color: #333;
  font-size: 18px;
  text-align: center;
}

JavaScript

JavaScript es un lenguaje de programación que permite agregar comportamiento e interactividad a una página web.

Se utiliza, por ejemplo, para:

  • responder a eventos como clics o pulsaciones de teclas
  • validar datos ingresados en formularios
  • modificar contenido y estilos sin recargar toda la página
  • comunicarse con servidores para obtener o enviar datos

En el contexto del desarrollo web inicial, JavaScript se ejecuta principalmente en el navegador. Junto con HTML y CSS forma la base del desarrollo del lado del cliente.

Aunque también existe JavaScript del lado del servidor, en este tema nos interesa sobre todo su uso en el navegador para construir interfaces dinámicas.
