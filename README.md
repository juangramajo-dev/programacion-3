# 🚀 Prog-III-Study-Guide

Una plataforma web interactiva y moderna diseñada como guía de estudio centralizada para la materia **Programación III** (UTN FRLR). Este proyecto transforma apuntes técnicos estáticos en una experiencia de aprendizaje dinámica con una estética *premium*.

![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Desarrollo-green)
![Licencia](https://img.shields.io/badge/Licencia-MIT-blue)

## ✨ Características Principales

*   **Dashboard Inteligente**: Interfaz principal basada en tarjetas con diseño *Glassmorphism* y animaciones de entrada fluidas.
*   **Menú Lateral "Hamburger"**: Navegación intuitiva y responsiva implementada puramente con CSS, asegurando compatibilidad y velocidad.
*   **Identidad Visual**: Logotipo personalizado construido con código (`</>`), integrando la esencia del desarrollo web.
*   **Modo Oscuro Premium**: Paleta de colores optimizada para largas sesiones de estudio (Tokyo Night inspirado).
*   **Contenido Interactivo**: Botones de copiado rápido para fragmentos de código y soporte para fórmulas matemáticas mediante MathJax.

## 🛠️ Tecnologías Utilizadas

*   **HTML5 Semántico**: Para una estructura sólida y accesible.
*   **CSS3 (Vanilla)**: Diseño responsivo, Flexbox, Grid, y animaciones personalizadas.
*   **JavaScript**: Manejo de animaciones por scroll (Intersection Observer API).

## 📂 Estructura del Contenido

El proyecto se divide en módulos clave para el desarrollo web:

1.  **🌐 Introducción a la Web**: Fundamentos de Internet y arquitectura Cliente-Servidor.
2.  **📄 Introducción a HTML**: Estructura básica y etiquetas esenciales.
3.  **📦 Bloque vs En Línea**: Comportamiento de elementos en el layout.
4.  **🎨 Introducción a CSS**: Selectores, cascada y diseño visual.
5.  **🔲 El Modelo de Caja**: Control de márgenes, bordes y espaciados.
6.  **🌿 Introducción a Git**: Fundamentos de control de versiones.
7.  **🔱 Ramas en Git**: Flujos de trabajo y branches.
8.  **⚡ Introducción a JavaScript**: Lógica e interactividad básica.

## 🚀 Cómo usar este proyecto

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/TuUsuario/Prog-III-Study-Guide.git
    ```
2.  Abre el archivo `index.html` en cualquier navegador moderno.
3.  ¡Explora los temas usando el menú lateral o el dashboard principal!
